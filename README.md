# Workout_helper
Designed to help me track my new rotational workout schedule, created out of the need to beat my current progression plateau.

## Project Overview
The system is comprised of the following components:
- Database (stores all exercise details)
- Kotlin mobile app
	- query calling backend
	- workout printing frontend 

## Technologies Used
- Antigravity
- Supabase database
- Kotlin
